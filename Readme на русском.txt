 Основные возможности приложения:

1. Автоматический кликер для игры в Blum боте в Telegram:
   - Распознавание элементов игры на экране на основе цвета пикселя
   - Автоматическое запуск следующей игры
   - Установка количества игр которые необходимо сыграть

2. Настройка через конфигурационный файл config.json:
   - Цветовые диапазоны для распознавания (до 10 цветов)
   - Задержки между кликами на кнопку "Играть"
   - Относительное положение кнопки "Играть" в процентах

3. Управление выполнением:
   - Пауза/возобновление по нажатию клавиши 'P'
   - Ограничение количества игр через параметр -play
   - Защита от многократных кликов

4. Безопасность:
   - !ВАЖНО! Требуется запуск от имени администратора для корректной работы с системными API Windows
   
---

 Пример запуска через командную строку (cmd):

1. Откройте меню "Пуск" и найдите "Командная строка" или "cmd"

2. Щелкните правой кнопкой мыши по "Командная строка" и выберите "Запуск от имени администратора"

3. В открывшемся окне перейдите в папку с программой, например:
   ```
   cd C:\Users\Username\Documents\Blum-bot
   ```

4. Запустите программу с указанием количества игр (например 5) через флаг -play:
   ```
   blum_bot.exe -play 5
   ```

   Где:
   - `blum_bot.exe` - имя исполняемого файла программы
   - `-play 5` - флаг, указывающий количество игр (в данном случае 5)

5. Дождитесь сообщения о начале работы и следуйте инструкциям в консоли.

---
 Важно:
- Если вы увидите ошибку о необходимости запуска от имени администратора, закройте программу и повторите шаги 1-4, убедившись, что cmd запущена с правами администратора.
